# Upload to pypi

1. Install twine

    ```bash
    pip install twine
    ```

2. Build Dist

    ```bash
    python3 setup.py sdist bdist_wheel
    ```

3. Check source

    ```bash
    twine check dist/*
    ```

4. Upload to TestPypi

    ```bash
    twine upload --repository-url https://test.pypi.org/legacy/ dist/*
    ```

5. Install

    ```bash
    sudo pip install -i https://test.pypi.org/simple/ youtube-stream -U
    ```
