# pymusic

## Requirements

- [Python](https://www.python.org/downloads/)
- [VLC](https://www.videolan.org/)

## Build and Run

- Install Python, pip and VLC
- Install youtube-stream

  ```bash
  pip install youtube-stream
  ```

- Run from command line (Powershell/Bash/CMD/zsh/fish)
  - for voice search

  ```bash
  stream main.py
  ```

  - Or, for text search

  ```bash
  stream <song name>
  ```

### Enjoy
