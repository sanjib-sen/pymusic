# pymusic

## Requirements:
- [Python](https://www.python.org/downloads/)

## Build
Install Dependencies through CMD/Powershell/Bash/ZSH
```bash
pip install bs4 speech_recognition pyaudio
```

## Run
```bash
python3 main.py
```
### for voice search
Or
```bash
python3 main.py <song name>
```
### for text search
